> flex-grow:1 el elemento ocupa todo el espacio libre
posible en el main-axis (en este caso el eje-y)
> flex-grow:2 el elemento ocupa el doble del espacio libre
> flex-shrink:1 el elemento se achica cuando NO
hay espacio libre (en el main-axis)
> flex-shrink:2 el elemento se achica el doble cuando NO
hay espacio libre (en el main-axis)
> flex-basis:200px el elemento tiene un tamaño por defecto
suponiendo que hay espacio libre (en el main-axis)
